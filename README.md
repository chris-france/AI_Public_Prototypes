# CascadeSwarm

AI/ML Proof-of-Concept Prototypes

A collection of 10 high-impact demos showcasing advanced reasoning, multimodal analysis, cost-optimized inference, and real-world engineering intelligence on private GPU hardware.

## Prototypes

1. **RFP → Winning Bid in 7 Minutes**  
   Parses complex RFPs, extracts scope, generates technical approach, staffing, risk matrix, and market-aligned pricing using historical data.

2. **CAD/Revit Booster**  
   Auto-extracts quantities, detects clashes, and identifies missing elements from BIM models to refine estimates and improve accuracy.

3. **Charleston Smart Bridge + Omniverse 4D**  
   Real-time 4D digital twin with physics simulation and AI-driven safety monitoring in NVIDIA Omniverse.

4. **60-Second Point Cloud → Perfect BIM**  
   Automatic registration and semantic segmentation of raw LiDAR scans into clean, usable BIM models.

5. **GPU BIM Compliance Checker**  
   Full IFC model validation (clashes, code compliance, constructability) executed on GPU in seconds.

6. **Inference Cost Dashboard**  
   Live dashboard comparing private GPU inference costs vs major cloud APIs, tracking tokens/sec, $/1M tokens, and cumulative savings.

7. **4090 Frontier**  
   70B–405B-class models running at 75–90 tokens/sec on a single consumer RTX 4090 (RTX 5090-ready).

8. **One-Click Private Cluster**  
   Bare metal or cloud console → fully tuned, hardened 8×GPU inference cluster deployed in under 9 minutes.

9. **Private Red Team Overlay**  
   Real-time cyber kill-chain detection and termination running on private GPU infrastructure.

10. **Lotus of 10 000 Suns**  
    Real-time sacred-geometry visualization of 10 000 dynamic suns at 300+ fps — a meditation on light, abundance, and emergence.

## Tech Stack
- Python + Streamlit / Gradio interfaces
- Open-source models (Llama-3.1, Mixtral, Gemma-2, Phi-3)
- Private GPU inference (RTX 4090/5090, RunPod, local clusters)
- NVIDIA Omniverse, CUDA, PyTorch, Open3D

## Notes
Prototypes 1–5 are internal AECOM-focused tools (video demos available under NDA).  
Prototypes 6–10 are public, open-source ready, and designed for maximum performance on consumer and cloud GPU hardware.
