# Prototype 01: RFP → Winning Bid in 7 Minutes

Internal AECOM tool for rapid proposal generation from complex RFPs.

## What It Does
- Ingests RFP documents (PDF/text)
- Extracts scope, requirements, and evaluation criteria
- Generates technical approach, staffing plan, risk matrix, and pricing
- Learns from historical wins for continuous improvement
